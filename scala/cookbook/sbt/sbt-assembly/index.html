sbt - 用 sbt-assembly 把相依性包起來

用 sbt-assembly 把相依性包起來

使用 Scala 的時候，通常會搭配 sbt 做建置工具。但 sbt 在建置時，其實只是將 source code 編譯包成 jar 檔而已，並不會把專案的相依套件包起來，這原理跟 maven 一樣。那麼，在 maven 裡，可將相依函式庫打包在輸出的 jar 檔裡，稱作 fat jar (或 uber jar)，在 sbt 裡做不做的到呢？

其實可以。

設定 plugin

在 project/plugins.sbt 裡，加上 sbt-assembly ，本文撰寫時，最新版是 2.0.0：

addSbtPlugin("com.eed3si9n" % "sbt-assembly" % "2.0.0")

設定 build

在 build.sbt 裡，加上 mainClass 及 assemblyJarName 兩項設定：

assembly / mainClass := Some("example.Hello"),
assembly / assemblyJarName := "hello.jar",

打包、執行

設定完成後，執行 sbt clean package，在 target/scala-{版號}/ 中，執行 scala jar hello.jar 即可。
